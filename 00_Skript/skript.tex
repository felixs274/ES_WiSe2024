\section{I/O Ports}

Ports werden durch 8 Bit Register konfiguriert.

\textbf{D}ata \textbf{D}irection \textbf{R}egister (DDR) \\

\vspace{0.5em} 

0 = Input 

1 = Output

\vspace{1em} 

\textbf{Default} Wert ist \textbf{0} bzw. \textbf{Input}.
Damit wird verhindert, dass ein Verbraucher ausversehen zu viel Strom zieht und damit den Mikrocontroller besch채digt.
